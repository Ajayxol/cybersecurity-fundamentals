
# 🛡️ Cybersecurity Fundamentals

A collection of **basic cybersecurity projects and exercises** designed to demonstrate fundamental concepts in cybersecurity.
This repository contains small Python-based tools and scripts that help understand common security practices such as password security, encryption basics, and vulnerability awareness.

The goal of this repository is to **learn, practice, and demonstrate core cybersecurity principles** through practical examples.

---

## 📌 Objectives

* Understand the **basics of cybersecurity**
* Learn how **security tools work internally**
* Practice **secure coding techniques**
* Build small **security-focused Python projects**

---

## 📂 Projects Included

### 🔐 Password Strength Checker

A Python program that evaluates the strength of a password based on security best practices.

**Features**

* Checks minimum password length
* Detects uppercase and lowercase letters
* Detects numbers
* Detects special characters
* Classifies passwords as **Weak, Moderate, or Strong**
